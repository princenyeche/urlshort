# URLShort

A URL Shortener in Go, testing the language for string interpolation.

## To resolve DIR issue
```
go mod tidy
```

## Running the application

Ensure that you are in the project directory
```shell
go run main.go
# OR
go run .
```

## About Go
- Go doesn't require an import statement within the same directory.
- Go doesn't require a go.mod file in every directory.
- Go doesn't require a main.go file in every directory.
- You can simply import a function the same directory in any of the files within the same directory.
